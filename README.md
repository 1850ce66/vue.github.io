# vue.github.io

這是個甜點電商的作品集，展示了店鋪的多種產品的訊息和購物功能,頁尾並附上聯繫方式。

## 目錄

- [功能](#功能)
- [使用技術](#使用技術)
- [使用](#使用)
- [聯繫方式](#聯繫方式)


## 功能

- 頁首包含導航菜單，提供各種頁面鏈接
- 產品展示區域，包括圖片和產品描述
- 甜點類別連接
- 當使用較小螢幕瀏覽時,選單會是的bootstrap中的漢堡選單
- 包含聯繫方法和版權聲明的頁腳
- jquery購物車功能
- rwd

## 使用技術

- HTML5
- CSS3
- javascrip
- Bootstrap 3.3.6
- jQuery 2.2.3
- rwd

## 使用
1.在網頁瀏覽器中打開 index.html 文件。
2.瀏覽產品頁面時，將商品加入購物車，並查看右上方的購物車圖示。
